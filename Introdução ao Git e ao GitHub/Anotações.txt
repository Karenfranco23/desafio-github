Como clonar um repositório:

1) Abrir a pasta onde será salvo o repositorio e clicar em "Git Bash Here";
2) Dar o comando git clone e em seguida colar o link do repositorio em HTTPS.


Para atualizar o repositorio com alguma pasta criada na maquina:

1) Dar um git status para ver qual é a nova pasta criada;
2) Dar um git add . ou git add -A para enviar essa pasta para o controle de versão local.

Para comitar um arquivo:
git commit -m "inclusao das anotações do curso Git/GitHub
